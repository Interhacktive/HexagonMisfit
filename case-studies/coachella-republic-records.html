The Brief
"We need visuals that feel as alive as the music — nothing pre-rendered, nothing canned. Everything live."

Republic Records brought Hexagon Misfit in to create the visual environment for their Coachella VIP party, featuring Phantogram. The brief was clear: fully live, fully reactive — no pre-rendered loops, no canned sequences.

The result was a real-time generative visual system built in TouchDesigner, driven entirely by audio analysis of the live set. Every beat, every frequency shift, every build in the music drove corresponding changes in the visuals — the room breathing with the music.

Scope of Work

  • Real-time audio analysis pipeline
  • TouchDesigner generative visual system
  • Multi-display output configuration
  • Live operation throughout event
  • Content direction and visual design
The Challenge

A live visual environment that has to match the energy of a Coachella set — in real time.

Live event visuals at Coachella carry a particular pressure: the crowd is experienced, the bar is high, and the music doesn't wait. A pre-rendered video loop is immediately apparent to anyone who's paying attention.

The system had to respond convincingly to the nuances of a live performance — not just the beat, but the texture, the dynamics, the moments of tension and release — without any human at the controls adjusting it in real time.

Our Approach

Four phases from brief to opening night.

01 — Audio Analysis

Frequency-Driven Pipeline

TouchDesigner's audio analysis tools were configured to extract frequency bands, amplitude envelopes, and transient events from the live audio feed. Each parameter drove distinct visual behaviors across the system.

02 — Generative System

Real-Time Visual Engine

A fully generative visual system was built that responded to the audio analysis data — geometry, color, texture, and motion all driven in real time. Nothing was pre-authored or pre-timed.

03 — Output Configuration

Multi-Display Integration

The system was output across multiple displays throughout the venue, creating an immersive visual environment that wrapped the space rather than being confined to a single screen.

04 — Live Operation

Full Event Coverage

Our team operated the system live throughout the event, monitoring system health and making macro adjustments as the set progressed — ensuring the visuals stayed in sync with the energy of the performance.

The Result

A room that breathed with the music — and a client relationship that continued from there.

The visual environment was received enthusiastically by the Republic Records team and Phantogram's crew. The fully live approach created a visual experience that felt unique to that specific set — something that couldn't be replicated or re-used.

"When the visuals are truly live, the crowd feels it — even if they can't say why."
