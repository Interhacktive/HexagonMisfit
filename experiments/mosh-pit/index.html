Mosh Pit

Two hundred and eighty individual letters, each with its own spring constant, damping coefficient, rotation bias, and scale ceiling. No two letters behave the same way when the cursor moves through them.

The six sliders change the physics in real time — pull chaos up for wild overshoot, push spring up for snappier response, drop scale down for a subtler crowd. All letters recompute their personality from the new parameters instantly.

Per-letter Physics Spring Integration Variable Fonts Live Config RAF Loop Range Sliders

// sliders

radius Cursor influence radius in pixels. Higher = more letters react at once.
spring Spring stiffness. High values snap letters back fast; low values are sluggish.
chaos Inverse damping. High chaos = low damping = overshooting, bouncing letters.
scale Maximum letter growth at full cursor activation.
rotate Maximum rotation in degrees. Each letter has a random direction bias.
weight Font-weight variation amplitude. High = heavy bold swells on activation.
