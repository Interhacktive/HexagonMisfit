Rain Text

ASCII rain falls through a block of monospaced text. Each raindrop displaces characters horizontally, sending ripple waves that push letters left and right before they spring back. Click anywhere to trigger lightning that illuminates the clouds and makes the text glow white.

The story is about rain. The medium is rain. The text reads while it weeps.

// controls

speed How fast the raindrops fall. Higher = heavier downpour.
density Number of concurrent raindrops. More drops = more ripples.
wind Horizontal wind. Negative = left, positive = right. Tilts the rain and drifts drops sideways.
dropsize Trail length behind each raindrop. Longer trails = heavier-looking rain.
elastic Spring return stiffness. High = snappy bounce back. Low = slow, lazy drift.
textsize Font size in pixels. Rebuilds the character grid — larger text means fewer characters but more presence.
click Click anywhere on the canvas. A lightning bolt forks down to the click point, illuminating the clouds and making all text glow.
